Blaze for WordPress allows you to export content from Blaze directly to your WordPress site.

== Description ==

Blaze revolutionizes the way you create content with its cutting-edge brand voice engine.  Blaze analyzes your writing style from every angle. In seconds, it figures out exactly what makes you sound like you.  With Blaze for WordPress you can generate content in your unique voice and then publish it directly to your WordPress site.

= How to get started =

- [Sign up for Blaze](https://blaze.ai?utm_source=integration&utm_medium=marketplace&utm_campaign=wordpress). (Blaze is free for 7 days, after which you can pick a plan that fits your usage-based needs.)
- Install the plugin, then head to [https://app.blaze.ai/](https://app.blaze.ai?settings=integrations) to get started.


== Installation ==

= Using the WordPress Dashboard =

1. Navigate to *Add New* in the plugin dashboard
2. Search for *Blaze for WordPress*
3. Click **Install Now**
4. Activate the plugin on the plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to *Add New* in the plugin dashboard
2. Navigate to the *Upload* area
3. Select the downloaded zip file of the plugin from your computer
4. Click **Install Now**
5. Activate the plugin on the plugin dashboard
